# Miniflux configuration

## Templates

Custom Miniflux CSS theme that auto-switches light/dark Catpuccin themes with system settings, plus a small JS helper to open articles in a new tab

### Usage


1. Copy the theme from [this](catpuccin_miniflux.css) directory
2. In miniflux, go to Settings > Application Settings
3. Paste the CSS into the `Custom CSS` text box at the bottom of the page
4. Copy the script from [this](new_tab.js) directory
5. Paste the JS into the `Custom JavaScript` text box underneath the previous one
6. Click `Update`
